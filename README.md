# Odin Recursion

Solving the classic Fibonacci and Merge Sort problems using recursion in JavaScript.

## Running the code
Execute the files directly using Node.js:
```bash
node fibonacci.js
node mergeSort.js
```

## Running the tests
This project uses Jest to verify both algorithms. 

1. Install the testing dependencies:
```bash
npm install
```

2. Run the test suites individually:
```bash
npm test fibonacci.spec.js
npm test mergeSort.spec.js
```
